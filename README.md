# Heritage Treasures: An In-Depth Analysis of UNESCO World Heritage Sites
This project presents a detailed analysis of **UNESCO World Heritage Sites** around the world. It explores the geographical distribution, categories, and conservation status of heritage locations. The aim is to understand how cultural and natural heritage sites are distributed globally and identify regions that require preservation efforts.
* Analyze the global distribution of World Heritage Sites.
* Identify endangered heritage locations.
* Study regional patterns of cultural and natural heritage.
* Visualize important insights using data analysis and charts.
The dataset used in this project contains information about:
* Name of the heritage site
* Country and region
* Year of inscription
* Type of site (Cultural, Natural, Mixed)
* Status (Endangered / Not Endangered)
* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Jupyter Notebook
The project includes the following analysis:
### 1. Distribution of World Heritage Sites

Analysis of the number of heritage sites in different countries and regions.

### 2. Cultural vs Natural Heritage Sites

Comparison of cultural, natural, and mixed heritage sites across the world.

### 3. Endangered Heritage Sites

Identification and analysis of heritage sites that are currently listed as endangered.

### 4. Regional Insights

Study of heritage site distribution across continents and their conservation importance.

The analysis highlights:

* Countries with the highest number of heritage sites.
* Regions with significant cultural heritage concentration.
* Locations where heritage sites are at risk and need protection.

World Heritage Sites represent valuable cultural and natural history. Protecting these sites is important for preserving global heritage for future generations. This project helps understand their distribution and the importance of conservation efforts.
